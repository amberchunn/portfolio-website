Meal Planning App

A meal planning site that allows you to create your meal plan for the next week from a list of stored recipes. Project was created for a small hackathon with Fabian Kaegy. The application uses the WordPress REST API to fetch recipes from a custom post type. Additionally, a custom Gutenberg block was created for quick recipe entry. Technologies used include, React, JavaScript, PHP, and SASS.
